# Agent Dashboard — Command Center

## What this is
Single-page HTML dashboard showing all agents, projects, orchestrator state, live services, Mac Mini M4 status, Linear issues, CodexBar usage, and comms bus. Auto-generated by a Python pipeline.

## Source files

| Path | Role |
|---|---|
| `~/scripts/build-agent-dashboard.py` | Thin entry point — calls `dashboard_builder` package |
| `~/scripts/dashboard_builder/` | Modular Python package (config, agents, projects, comms, linear, metrics, codexbar, orchestrator, live, html_builder, md_builder) |
| `~/scripts/dashboard-assets/style.css` | Extracted CSS |
| `~/scripts/dashboard-assets/script.js` | Extracted JS (sidebar nav, mode slider, local services probe) |

## Output

| File | Description |
|---|---|
| `~/agent-dashboard.html` | Full HTML dashboard (opened in browser) |
| `~/agent-dashboard/index.html` | Git-tracked copy for GitHub Pages |
| `~/ObsidianVault/90-Operations/second-brain/reports/YYYY-MM-DD -- agents-dashboard.md` | Obsidian markdown snapshot |

## Data sources

- **Projects**: scanned from `~/ObsidianVault/20-Projects/*/Tech-Base/TODO.md`
- **LUMUS**: parsed from `~/ObsidianVault/LUMUS-CONTEXT.md`
- **Comms**: `~/.agent-bridge/comms.jsonl`
- **Live feed**: `~/scripts/live-feed.json` (generated by `generate-live-feed.py`)
- **Health**: `~/.agent-bridge/health/*.json`
- **Orchestrator**: `~/.agent-bridge/orchestrator.json`
- **Usage**: `~/.agent-bridge/usage_snapshots.json`
- **Token ledger**: `~/.agent-bridge/token_ledger.jsonl`
- **CodexBar**: `~/Library/Group Containers/group.com.steipete.codexbar/widget-snapshot.json`
- **Linear**: via `~/.claude/lib/linear_api.py` CLI
- **JARVIS Pipeline**: `/Users/pirajoke/jarvis` git checkout, `launchctl` service state, `/tmp/jarvis-bot.err` AIOS startup log, and `/Users/pirajoke/jarvis/data/jarvis.db` `answer_sources`

## Pipeline

```
generate-live-feed.py  →  live-feed.json
                              ↓
build-agent-dashboard.py  →  agent-dashboard.html + agents-dashboard.md
                              ↓
dashboard-rebuild.sh      →  copy to ~/agent-dashboard/index.html + git push
```

Both scripts run as launchd services (`com.pirajoke.generate-live-feed`, `com.pirajoke.dashboard-server`).

## How to rebuild manually

```bash
cd ~/scripts && python3 build-agent-dashboard.py
```

## Module structure

```
dashboard_builder/
├── __init__.py
├── config.py          # Paths, constants, agent defs, colors, SVG avatars
├── agents.py          # Health logic, network SVG
├── projects.py        # Scan projects, count TODOs, parse LUMUS
├── comms.py           # Read comms bus, chat-style HTML
├── linear.py          # Fetch Linear issues, render HTML
├── metrics.py         # Load agent metrics, render leaderboard
├── codexbar.py        # CodexBar provider usage bars
├── orchestrator.py    # Mode slider, subscriptions, budgets
├── live.py            # Live feed, events, health, Mac Mini panel
├── jarvis_pipeline.py # JARVIS runtime/deploy/source-aware card
├── html_builder.py    # Full HTML assembly
└── md_builder.py      # Obsidian markdown output
```
